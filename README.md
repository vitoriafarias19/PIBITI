PLATAFORMA DE CONEXÃO ENTRE PROFESSORES E ALUNOS PARA OPORTUNIDADES DE PIBIC E PIBITI

Este projeto é um sistema web para gerenciamento de vagas de PIBIC/PIBITI da Universidade Federal do Amazonas (UFAM).
O sistema permite que professores publiquem vagas e que alunos se candidatem, seguindo regras administrativas definidas pela instituição.
